# 🐪 Perl Mobile Setup & Auto-suggestions Guide

Get a full Perl development environment on your phone using CodeDroid and Termux.

---

## 🛠️ Step 1: Install Perl
Open Termux and run the following command to install the Perl runtime:

```bash
pkg install perl
```

Verify the installation:
```bash
perl -v
```

---

## 🧠 Step 2: Enable Language Support
CodeDroid runs Perl scripts natively using the `perl` binary in Termux.

---

## ✨ Step 3: Enable Auto-suggestions
1. Start your CodeDroid API server.
2. Open CodeDroid Web IDE and create/open any `.pl` file.
3. Start typing (e.g., `print "Hello\n";`) and run your Perl code!
